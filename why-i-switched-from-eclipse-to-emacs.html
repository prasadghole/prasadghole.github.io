Why I switched from Eclipse to Emacs

was the my favourite IDE for past 10 years. It has everything a developer wants - Cross platform - With perspective can be used for any programming languages - Free - All the plug ins you want - Plenty of customisation

But then as it provides boost to your productivity you soon may overtake the features it support. There are many reasons for this - Speed speed speed Yes. Eclipse is great but with so many features it started feeling like blotted software. I mean with heavy burden on RAM you can get same performance with emacs. Partially slow reason goes to java.

Why emacs

I gave up on for simple windows tool notepad++ and sublime. But with more command line tools I started using I started to feel more of using emacs. Its very ineffective way to shift constantly between keyboards and mouse for simple copy paste operation. File opening with windows file open dialogue box is also bad.

##Let journey begin Remember a thousand mile journey starts with single step forward. But with Emacs single step is too hard to take. Starting from installation to simple file open, cut paste operations seems big. This is my third attempt for starting journey of emacs.But I will tell you. It's worth try. Once you push hard rewards are overwhelming .
