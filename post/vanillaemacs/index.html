Moving to Vanilla Emacs

May 26, 2023

Burn the bridge.

Instead of keeping spacemacs and classical Emacs I have decided to burn the bridge by deleting the spacemacs configuration directory. (off-course I have the backup kept somewhere safe)

Starting Fresh

Default basic setup

Deleting the spacemacs Emacs.d file and restarting the Emacs started the vanilla Emacs. As a noob on vanilla Emacs I will be using customization instead of solely depend on text configuration.

Startup configuration 

(setq initial-scratch-message nil)
(tool-bar-mode -1)
(menu-bar-mode -1)
(toggle-scroll-bar -1)

Package managers 

(require 'package)
(setq package-archives '(("melpa" . "https://melpa.org/packages/")
                         ("org" . "https://orgmode.org/elpa/")
                         ("elpa" . "https://elpa.gnu.org/packages/")))

Evil mode

One of the important reason for spacemacs was evil mode. Hence first thing was to enable it after 

(require 'evil)
(evil-mode 1)

How can Evil complete without mapping emacs exit current operation command by Escape key.

(global-set-key (kbd "<escape>") 'keyboard-escape-quit)

Line Numbers 

(global-display-line-numbers-mode 1)

Recent files (Vim Startify) 

(recentf-mode 1)
(setq recentf-max-menu-items 25)
(setq recentf-max-saved-items 25)
(global-set-key "\C-x\ \C-r" 'recentf-open-files)
