OTG

USB OTG is a dual core device which can act as both device and host. HS (High speed) FS (Full speed) 1. 1.25 Kbytes of dedicated RAM for advance FIFO control 2. Maximum USB bandwidth of one frame (1 ms ) without system intervention.

Clock

It receives the 48 MHz clock from RCC.

Writing in peripheral mode

CPU write 32 bit word to OTG push register. This data then directly transferred to Tx data configured within USB data RAM.

Reading in peripheral mode

CPU receives the data from USB by reading 32 bit pop register. This data is retrieved from a shared Rx FIFO.

Interrupt reception

Application will keep on receiving the Rx FIFO non-empty interrupt as long as there is one packet available to download.

Performance

  1. RAM Size
  2. FIFO size
  3. Quick access to AHB push/pop registers.
